# Sample of API testing automation

## Overview

This project contains a Robot Framework test suite to automate the generation of authentication token of the RESTful Booker API. The test suite generates a valid authentication token using correct credentials and validates the API response.

It demonstrates basic API automation skills including session handling and JSON validation.

### Prerequisites

Before running the tests, ensure you have:

- Python 3.x installed

- Robot Framework installed

- RequestsLibrary installed (pip install robotframework-requests)

### Installation

- Clone the repository:

- Install required libraries:
    - pip install robotframework
    - pip install robotframework-requests


### How to Run
robot restfulBooker.robot
