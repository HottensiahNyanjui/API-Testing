# QA Assessment

## Overview
This repository contains a Postman collection, API testing, a conceptual test plan for a payment gateway and screenshots of API testing.

## Repository Structure
- Postman/ → Exported Postman collection
- Documents/ → Test documentation for section A and section B
- screenshots/ → Evidence of executed test cases

## How to Run API Tests
1. Open Postman
2. Import the collection from the `Postman/` folder
3. Execute the requests in the provided order
